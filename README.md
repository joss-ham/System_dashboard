# System_dashboard

### This is a real time system health dashboard similar to windows task manager ###

I wanted to learn how to use the psutil and have made this simple project to toy around with it

## Tech Stack: ##
- **Language:** Python 3.11
- **Backend:** Flask
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Chart.js
- **DevOps:** Docker, psutil
- **Data:** JSON API Architecture

---

## prerequisites ##
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed and running.
